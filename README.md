# TA-zero.github.io
此项目由ai编写

主要用于测试ai编写代码能力

咕咕嘎嘎!!!

主要就是一个小游戏
原型是根据谷歌没网的时候的小恐龙游戏
由claude,deepseek共同编写
可能有部分逻辑不完整,有时间修复
由于全部堆到了一个文件,运行时释放技能会导致卡死
见谅

游戏背景:
无背景,ai写出来的能有啥背景

游戏玩法
控制一只长得和茄子一样的恐龙
通过跳跃和发射按钮来躲避障碍物以及后续出现的敌人
拥有火焰(角色)等级设定每提升10级伴随一次技能的升级
增益buff ：
一些小光球
红色：无敌20秒
蓝色：火焰伤害加强
黄色：分数增加200
彩色：血量回满，生命上限增加50，伴随三秒无敌，增加分数
白色：获得可以抵御伤害的屏障，共三个
金色：获得无敌20秒，穿透加强，伤害加强，同时增加生命恢复，分数获取速度提升4倍，获得三个屏障
